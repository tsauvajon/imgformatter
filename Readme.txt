Cet executable effectue les opérations suivantes :
1- Ouvre les images .jpg dans le dosser "in"
2- Les convertit en .png et les palces dans le dossier "out"
3- Les réduit avec une hauteur max de 160 px et les place dans "resized"
4- Optimize leur taille et les place dans "compressed"

Lancez simplement "jpgtopng.exe" pour lancer les traitements
